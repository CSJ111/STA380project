Specifically, Generative AI was utilized for the following purposes:

- Code Assistance & Debugging: We used AI to help generate the initial boilerplate code for the R Shiny web application interface (ui and server structure). AI was also used to help identify and troubleshoot specific coding errors (e.g., handling NA values and data type parsing issues within the dataset).

- Formatting & Visualization: AI assisted in providing syntax examples for generating the permutation distribution plots using ggplot2 and formatting the summary tables.

- Writing & Proofreading: AI tools were occasionally used to check grammar, refine the phrasing of our project proposal, and improve the clarity of our documentation.

- In terms of debugging, we described the UI and server code to the AI, and it provided suggestions on how to handle specific errors we encountered (e.g., issues with NA values in the dataset, or problems with data type parsing). We then implemented those suggestions and tested the application to ensure it was functioning correctly.
